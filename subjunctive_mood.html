가정법 1. 가정법 과거/과거완료/현재/미래 1) 가정법 과거 - 현재 사실 반대, 실현 불가능한 일 If S 과거동사, S + would/could/might... + 동사원형 if절의 be동사 과거형은 무조건 were을 쓴다(수일치 관련) 2) 가정법 과거완료 - 과거에 이미 발생한 사건에 대한 반대 If S had p.p., S + would/could/might... + have p.p. 3) 가정법 현재 - '조건문', 실현가능 If S 현재동사, S 조동사 원형(will/can/may...) + 동사원형 , 명령문(Please 동사원형) 4) 가정법 미래 - 가정법 현재에 비해 실현 가능성 없음 강조! If S should 동사원형, S 조동사원형(will/can/may...) + 동사원형 , 명령문(Please 동사원형) If I were you, I wouldn't do such a stupid thing. If we had placed the order earlier, we could have received it on time. If you should have any questions, please do not hesitate to contact us. (if절의 조동사는 'should'뿐이다, could가 없어도 말이 된다?) 2. If가 없는 가정법 : 가정법 도치 도치구문 : (주어와 동사의) 위치가 바뀐 문장 If가 생략도치유발인자 -> if절의 주어와 첫번째 조동사/be동사가 도치도치 1) 가정법과거 - If 주어 were + ~ -> Were S ~ 2) 가정법 과거완료 - If 주어 had p.p. -> Had S p.p. ~ 3) 가정법 미래 : If S should 동사원형 -> Should S 동사원형 ~ If you should have any questions, please do not hesitate to contact us. -> Should you have my questions, please do not hesitate~.
